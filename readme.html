WooCommerce Popup Notifications

Description: A WordPress plugin designed to enhance the shopping experience on WooCommerce-powered websites by displaying a notification in a popup whenever a product is added to the cart.

Features

  • Customizable Popup Notifications
  • Adaptive Display Positions
  • Product Image Handling
  • Extensible through Filters

Classes

WC_Popup_Notifications

Main class responsible for initializing the plugin's functionality. It handles the enqueuing of scripts and styles, generates the popup notification fragment, and listens for add-to-cart actions.

WC_Popup_Notifications_Admin

Handles the backend configuration for the plugin.

Filters

wc_popup_notifications_close_after_seconds: Allows changing the auto-close duration of the popup notification.

wc_popup_notifications_top_offset: Customizes the top offset for the popup when displayed at the top of the page.

Installation

  1. Upload the plugin files to the /wp-content/plugins/woocommerce-popup-notifications directory, or install the plugin through the WordPress plugins screen directly.
  2. Activate the plugin through the 'Plugins' screen in WordPress.
  3. Use the plugin's settings page to configure the plugin.

Usage

Once activated, the plugin works automatically by displaying a popup notification whenever a product is added to the cart. Administrators can customize the plugin's behavior through the WordPress admin panel.
